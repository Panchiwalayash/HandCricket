# Hand Cricket Game

A bright, minimal ZPL-inspired **Hand Cricket** web application built with Next.js. It features a stadium daytime stands background, simulated real-time squad opponents, dynamic big screen scoreboard widgets, and full-screen warning/strobe overlays for boundaries and wickets.

## Features
- **Direct Start**: Launches straight into the playing dashboard on page load.
- **Stadium Digital LED Scoreboard**: Tracks Player 1 (You) and Player 2 (Opponent) runs, deliveries, and selections.
- **Simulated Multiplayer Feed**: Delay action selection for 2 seconds to simulate a real-time multiplayer teammate challenge.
- **Big Moment Overlays**: Vivid visual flashes for boundaries (4s & 6s) and wickets.
- **Card Selection Clearing**: Clean card grid auto-resets selected outlines 1.5 seconds after a delivery is completed.

## Tech Stack
- Framework: Next.js (App Router)
- Styling: Vanilla CSS

## Development
To run the server locally:
```bash
npm run dev
```

To build for production:
```bash
npm run build
```
